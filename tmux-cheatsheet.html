⌨ TMUX CHEAT SHEET

vim-style config · prefix is Ctrl-a · ~/.tmux.conf
Prefix = Ctrl + a — press this before every command below

Panes — Navigation

prefix hMove left
prefix jMove down
prefix kMove up
prefix lMove right
Also works: click any pane with the mouse.

Panes — Splitting

prefix |Split horizontal (side by side)
prefix -Split vertical (top / bottom)
prefix xClose current pane
prefix zToggle pane zoom (fullscreen)
New panes open in the same working directory.

Panes — Resizing

prefix HGrow left (5 cells)
prefix JGrow down
prefix KGrow up
prefix LGrow right
Repeatable — hold prefix, tap direction multiple times. Also: drag borders with mouse.

Windows

prefix cNew window
prefix nNext window
prefix pPrevious window
prefix 1–9Jump to window #
prefix ,Rename window
prefix &Close window
Windows are numbered starting at 1.

Copy Mode (vi keys)

prefix EnterEnter copy mode
vBegin selection
yYank (copy) & exit
EscapeCancel / exit copy mode
/Search forward
?Search backward
Navigate with h/j/k/l, Ctrl-u/d for half-page scroll. Mouse scroll also enters copy mode.

Sessions

prefix dDetach from session
prefix sList / switch sessions
prefix $Rename session
prefix rReload tmux.conf
prefix d detaches — the session keeps running in the background.

Command Line

tmux new -s nameCreate a new named session
tmux attach -t nameReattach to a running session
tmux lsList all sessions
tmux kill-session -t nameKill a session
tmux kill-serverKill tmux and all sessions
